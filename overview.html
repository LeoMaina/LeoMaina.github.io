Machine Learning

60%

Most of the machine learning projects I have done involved the use of Tensorflow Object detection API where I used transfer learning to build models and train them on the unique datasets for each project. For instance, the Wheel Clamping Robot[Patented and documented on the projects tab] utilised a raspberry pi camera as input to a model that could detect cars and the position of the wheels in real time. I have also done a freelance project on Audio classification using python where the model built was trained to identify a specific beep sound in any audio file and only actuate commands when this beep sound is detected. For this I had to use libraries such as librosa.
